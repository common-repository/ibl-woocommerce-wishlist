Using IBL Wishlist is display Customizable Wishlist Button and Product Added into the Wishlist page.

== Description ==
WooCommerce Wishlist is a tool that help you to convert your site users into customers. When customers can’t buy a product at this time. Possibility to save products for later encourages users to return to your site and make a purchase. Users can share their wishlist with close people. By sharing their Wishlists, they are bringing you, new potential customers.

> **Plugin Features**
>
> * Add Name of Wishlist.
> * Remove Product From Wishlist.
> * Set "Add Wishlist" button Text.
> * Set "View Wishlist" button Text.
> * The position of Wishlist button.
> * Shortcode for “Add wishlist” button
> * Display as a link or button.
> * Display icon or text with an icon.
> * Set icon with a custom color.
> * Share Wishlist on Social Network (Facebook, Twitter, Google+, Pinterest, Email)
> * Fully Customizable Appearance
> * Responsive & Retina Ready
> * Translation ready●WPML Support
> * Product variations Support


> **Pro Features**

> * Wishlist privacy settings

> * Option for use active theme style.
> * Change the button background color.
> * Change the button text color.
> * Change border color.

== Installation ==
1.	Install the IBL Wishlist plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the /wp-content/plugins/ directory).

2.	Activate the plugin through the Plugins menu in WordPress admin.

3.	Customize Wishlist Settings From IBL Wishlist menu in WordPress admin.

4.	Show Wishlist Button to the Each Product Page and Product List. Also Plugin will Create Wishlist Page.
